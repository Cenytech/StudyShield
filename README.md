# StudyShield
A web extension that keeps you focused on your study, you cannot stop the timer, you can add a custom URL that distracts you to help you focus.

## 🛠️ How to Install (Chrome & Brave)

Since this extension is not yet hosted on the official Web Store, you can easily load it manually using **Developer Mode**:

### Step 1: Download the Extension
1. Click the green **Code** button at the top right of this GitHub page.
2. Select **Download ZIP**.
3. Once downloaded, extract (unzip) the file to a folder on your computer that you won't accidentally delete.

### Step 2: Load it into your Browser

#### For Google Chrome:
1. Open Chrome and type `chrome://extensions/` into the URL bar.
2. In the top-right corner, turn ON the **Developer mode** toggle.
3. Click the **Load unpacked** button that appears in the top-left.
4. Select the folder you just unzipped (the folder containing your `manifest.json` file).

#### For Brave Browser:
1. Open Brave and type `brave://extensions/` into the URL bar.
2. In the top-right corner, turn ON the **Developer mode** toggle.
3. Click the **Load unpacked** button in the top-left.
4. Select the folder you just unzipped.

---

🎉 **You're all set!** The extension icon should now appear in your browser's toolbar.
